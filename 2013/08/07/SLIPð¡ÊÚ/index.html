SLIP协议

Posted on 2013-08-07   |   In 通信   |  

SLIP协议（Serial Line Interenet Prococol)串行网络协议。RFC1055

早期互联网时代，还没有发明以太网双绞线这种可以传输查分信号的东西，传输网络信号的介质有限。所以最早的Internet是通过串口来传输的。从今天的角度来看，当年的互联网真的是慢的可怜，工业界用的的串口速度很多都在115200字节每秒。但是这样的速度在当年已经可以实现一些最最基本的基于文本的互联网。

要通过串口实现上网，一个很重要的问题是串口得有能力区分包。因为串口本身是一个二进制字节流，并不提供任何协议，基于分包的互联网在串口上传输，怎么在接收端知道收到的是一个一个包呢。SLIP协议就是用来解决这个问题的。

首先为了实现分包，每个包都必须要求有一个包头，可以让接收端识别这个包头，才知道要开始接收的是一个包。而在SLIP协议里就用了0xC0这个字节专门规定为包头（也同时为包尾）。

那么问题又来了，要是规定了这个字节当包头，那么发送端就在每个包前面加上这个字节，但是因为数据流里面是任何数据都有可能存在的，要是数据流里本身就包含了0xC0的话，接收端就会分错包。SLIP里的解决办法是发送端只要碰到0xC0就把它转换成两个字节0xDB+0xDC，并且把0xDB也定义成特殊字符，否则的话碰到0xDB又会当做数据帧处理，每当碰到0xDB就替换成0xDB+0xDD。做过这样的处理之后在接收端只要收到0xC0就肯定知道是包头，然后收到0xDB的时候要等到下一个才知道是什么数据，再把两个数据一起还原成0xC0或者0xDB。

RFC1055中已经给出C的简单实现。这里给出Java的实现版本。

private static final int END = 0xC0;
private static final int ESC = 0xDB;
private static final int END_ESC = 0xDC;
private static final int ESC_ESC = 0xDD;

/**
 * Send a packet
 * 
 * @param packet
 */
private static void send(byte[] packet) {

    sendChar(END);

    for (int i = 0; i < packet.length; i++) {
        switch (packet[i] & 0xFF) {
        case END:
            sendChar(ESC);
            sendChar(END_ESC);
            break;
        case ESC:
            sendChar(ESC);
            sendChar(ESC_ESC);
            break;
        default:
            sendChar(packet[i]);
            break;
        }
    }
}

/**
 * Receive a Packet
 * @return packet bytes 
 * @throws Exception
 */
private static ArrayList<Integer> receive() throws Exception {
    ArrayList<Integer> recvData = new ArrayList<Integer>();


    boolean endFlag = false;        
    while(true){
        int data = getChar();

        switch(data){
        case END:
            if(endFlag){
                //Packet END found
                endFlag = false;
                return recvData;
            }
            else{
                endFlag = true;
                break;
            }

        case ESC:
            int c = getChar();
            switch(c){
            case END_ESC:
                recvData.add(END);
                break;
            case ESC_ESC:
                recvData.add(ESC);
                break;
            default:
                //After ESC, no conventional byte detected
                throw new Exception("Receive Error");
            }
        default:
            if(endFlag){
                recvData.add(data);
            }
            break;    
        }
    }

}
